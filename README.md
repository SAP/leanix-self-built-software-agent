# leanix-self-built-software-agent

## About this project

This is an automated software discovery system that uses AI agents to analyze GitHub repositories and identify self-built software running in production environments. Core Purpose The project aims to discover and catalog self-built software without human intervention by: Automatically analyzing GitHub repositories in an organization Classifying repository types (services, libraries, infrastructure, etc.) Identifying owners, dependencies, technology stacks, and runtime details Storing all findings in a structured database for reporting
